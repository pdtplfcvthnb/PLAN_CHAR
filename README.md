# Маршрут по зданию колледжа Царицыно(3 этажа)

Веб-приложение для построения маршрута между любыми аудиториями трёхэтажного здания (находящимися на одном этаже)
## Функционал
- Переключение между этажами
- Построение маршрута от любой точки к любой
- Маршрут проходит только по коридорам и лестницам
- Визуальное отображение маршрута на схеме

## Как использовать
1. Выберите начальную точку
2. Выберите конечную точку
3. Нажмите "Построить маршрут"
